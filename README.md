# OpenJFX 21 for FreeBSD

This project ports [OpenJFX 21](https://github.com/openjdk/jfx21u) to the FreeBSD (x86-64) platform.

The ported version has been published to Maven Central:

- [`org.glavo.openjfx:javafx:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx/21.0.11-freebsd)
- [`org.glavo.openjfx:javafx-base:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx-base/21.0.11-freebsd)
- [`org.glavo.openjfx:javafx-graphics:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx-graphics/21.0.11-freebsd)
- [`org.glavo.openjfx:javafx-controls:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx-controls/21.0.11-freebsd)
- [`org.glavo.openjfx:javafx-fxml:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx-fxml/21.0.11-freebsd)
- [`org.glavo.openjfx:javafx-media:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx-media/21.0.11-freebsd)
- [`org.glavo.openjfx:javafx-web:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx-web/21.0.11-freebsd)
- [`org.glavo.openjfx:javafx-swing:21.0.11`](https://central.sonatype.com/artifact/org.glavo.hmcl.openjfx/javafx-swing/21.0.11-freebsd)

All JARs use the `freebsd` classifier.

SDK and JMOD files can be downloaded from [GitHub Releases](https://github.com/Glavo/jfx21-freebsd/releases).
